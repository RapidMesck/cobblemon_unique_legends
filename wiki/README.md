# Unique Legends Wiki

Unique Legends is a server-side Cobblemon addon that makes selected Pokemon unique per server.

By default, the mod tracks:

- Legendary Pokemon
- Mythical Pokemon
- Ultra Beasts

When a tracked Pokemon is captured, that species becomes locked. Other players can still battle future spawns of that species, but they cannot capture them while the lock is active.

## Server-side only

The mod is designed to run on the server. Players do not need to install it on their clients.

Because of that, the mod blocks captures on the server instead of modifying the Cobblemon client battle UI.
